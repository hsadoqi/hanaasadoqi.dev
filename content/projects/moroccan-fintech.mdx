## Platform foundation

This project is about separating shared financial infrastructure from product-specific workflow behavior. The shared layer should own tenant identity, permissions, audit events, document metadata, and workflow status.

  label="Shared fintech platform primitives"

## Boundary rule

Shared primitives are extracted only when multiple products need the same contract. Product-specific behavior stays local until reuse is real.
