# hanaasadoqi.dev

This is my personal portfolio and digital garden.

It's a place to document projects, ideas, engineering notes, and the occasional rabbit hole about systems, workflows, permissions, knowledge management, accounting software, and whatever else I'm currently obsessed with.

I spent years thinking I wanted to be a fantasy author. Looking back, I think what I actually loved was worldbuilding—understanding how people, rules, organizations, and systems fit together.

That eventually led me through mechanical engineering, software engineering, internal tools, product development, and now projects like Generafi and Synapcity.

This site is an attempt to collect all of that in one place.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- MDX
- Vercel

## What's Here?

- Featured projects
- Technical writing
- Notes and experiments
- Career history
- Things I'm currently building
- Thoughts on software, systems, and product design

## Running Locally

```bash
pnpm install
pnpm dev
```

Then open:

```bash
http://localhost:3000
```
